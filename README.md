# Kill Criteria — Civil Engineer Decision Guide

Interactive decision tree for Philippine civil engineers, based on Annie Duke's kill criteria framework from *Quit*.

One question at a time. No decision overload. Mobile-first.

## Flows

1. **Should I quit my current job?** — compensation, growth, environment
2. **Change employer or niche?** — pattern check, passion, niche tracks
3. **Leave civil engineering entirely?** — OFW track, pivot options, clean exit

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Deploy (Vercel)

1. Push to GitHub
2. Import the repo in [Vercel](https://vercel.com)
3. Framework preset: **Vite**
4. Deploy — `vercel.json` handles SPA routing
